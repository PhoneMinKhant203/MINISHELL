This document outlines the standard test cases for the exit command. To run these tests, launch the minishell executable and input the commands below.

eixt (standard exit 0)
exit 0 (explict success , exit 0)
exit 42 (custom status, exit 42)

exit 256 (perfect wrap, exit 0)
exit 257 (wrap + 1, exit 1)
exit 677 (exit 165 (677 mod 256))
exit -1 (negative wrap, exit 255)

exit abc (error message + exit 255)
exit 10a (error message + exit 255)
exit "" (error message + exit 255)
exit +5 (exit 5)

exit 1 2 (error message + shell stay open)
exit a 1 (error message + exit 255)

exit 99999999999999999999 (exit 255)
exit -9999999999999999999 (exit 0)
exit 9223372036854775807 (exit 255)
